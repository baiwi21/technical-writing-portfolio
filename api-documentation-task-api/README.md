# Task Manager System API

This is a REST API for managing tasks in the Task Manager System.

------------------

## 🌐 Base URL
https://api.taskmanager.com

---

## 🔐 Authentication

All requests require a Bearer token:

------------------

## 📌 Endpoints

------------------

### POST /tasks

Creates a new task.

#### Request Body

```json
{
  "title": "New task"
}
