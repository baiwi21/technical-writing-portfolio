# Task Manager App – User Guide

## Overview
This guide explains how to use the Task Manager application.

---

## Getting Started

### Step 1: Create an account
Click "Sign Up" and enter your email and password.

### Step 2: Log in
Use your credentials to access the system.

---

## Creating a Task
1. Click "New Task"  
2. Enter task title  
3. Save task  

---

## Managing Tasks
- Edit task  
- Delete task  
- Mark as completed  

---

## FAQ

**Q: Can I recover deleted tasks?**  
A: No, deleted tasks cannot be restored.
